Questões sobre Computação na Nuvem

1. O que é a Computação na Nuvem?

A Computação na Nuvem é um modelo que permite o acesso a recursos de computação, como servidores, armazenamento e aplicativos, pela internet. Em vez de depender de hardware local, os usuários podem utilizar serviços e recursos oferecidos por provedores de nuvem, pagando conforme o uso. Para mais informações, você pode assistir ao vídeo: O que é Computação na Nuvem?

2. Quais as características da computação em Nuvem?

  • Escalabilidade
  • Elasticidade
  • Pagamento conforme o uso
  • Accessibilidade
  • Gerenciamento simplificado

3. Quais as modalidades de serviços? Dê, pelo menos, três exemplos de cada.

  • Infraestrutura como Serviço (IaaS):
    • AWS EC2
    • Google Compute Engine
    • Microsoft Azure Virtual Machines
  • Plataforma como Serviço (PaaS):
    • Google App Engine
    • AWS Elastic Beanstalk
    • Microsoft Azure App Services
  • Software como Serviço (SaaS):
    • Google Workspace
    • Microsoft Office 365
    • Salesforce

4. Quais os tipos de nuvem? Defina cada uma delas.

  • Nuvem Pública: Recursos são oferecidos por terceiros e compartilhados entre vários usuários. Exemplos incluem AWS, Google Cloud e Azure.
  • Nuvem Privada: Recursos são dedicados a uma única organização. Pode ser gerida internamente ou por um terceiro.
  • Nuvem Híbrida: Combina nuvens públicas e privadas, permitindo que dados e aplicativos sejam compartilhados entre elas.

5. Faça um Breve Resumo sobre os grandes provedores de Cloud: AWS, AZURE e Google Cloud.

  • AWS (Amazon Web Services): É a plataforma de computação em nuvem mais amplamente adotada e oferece uma ampla gama de serviços, incluindo computação, armazenamento, banco de dados e aprendizado de máquina. É conhecida por sua escalabilidade e ampla rede global de data centers.
  • Microsoft Azure: É a plataforma de nuvem da Microsoft que oferece uma vasta gama de serviços como computação, análise, armazenamento e rede. É integrada com muitos produtos Microsoft e é conhecida por sua compatibilidade com ambientes híbridos.
  • Google Cloud: Oferece serviços de computação, armazenamento e análise com um foco em big data e machine learning. É conhecida por suas capacidades de análise de dados e infraestrutura escalável.
